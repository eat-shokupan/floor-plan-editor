2. 部屋の追加・編集

部屋をドラッグで追加

  1. 左サイドバーの 「部屋追加」 をクリック（カーソルが十字に変わります）
  2. 描画エリア上で クリック＆ドラッグ して矩形を描きます
  3. ドラッグ中、リアルタイムでサイズ・帖数・mm寸法が表示されます
  4. マウスを離すと部屋が確定します

プリセットから追加

左サイドバーの 「プリセット」 セクションに LDK・和室・洋室・浴室・トイレ・玄関・廊下 などの定型部屋が用意されています。クリックするだけで追加できます。

部屋の選択

  1. 左サイドバーの 「選択」 ツールにする（デフォルト）
  2. 部屋をクリックすると選択状態になり、青い枠と 8つのハンドル が表示されます
  3. 右パネルに名前・色・サイズ・帖数などのプロパティが表示されます

部屋の移動

選択ツールで 部屋の内側をドラッグ すると移動できます。1/4 マス単位でスナップします。

部屋のリサイズ

選択すると表示される 8つの角・辺ハンドル をドラッグしてサイズ変更できます。

部屋の削除

  • 部屋を選択して Delete キー
  • または右パネルの 「削除」 ボタン

名前・色の変更

部屋を選択した状態で、右パネルの 「名前」 欄を編集したり、「色」 のカラーピッカーで色を変えられます。

3. L字・コの字型の部屋

1つの部屋は複数の矩形パーツを組み合わせて作れます。

作り方

  1. まず普通に 矩形の部屋を1つ作成 して選択します
  2. 右パネルの 「パーツ追加」 ボタンをクリックします
  3. カーソルが十字に変わるので、既存の部屋に隣接する位置 に新しい矩形をドラッグして描きます
  4. 確定すると部屋がL字型になり、帖数は自動で合算 されます
【L字型の例】 【コの字型の例】 ┌──────┐ ┌───┐ ┌───┐ │ Part 1 │ │ P1 │ │ P3 │ │ 4×3 │ │ │ │ │ │ ├───┐ │ ├──┤ │ │ │ Part 2 │ │ │ P2 │ │ └──────┤ │ └───┤ ├───┘ └────┘ └───┘ 帖数 = Part1 + Part2 の合計 帖数 = P1 + P2 + P3 の合計
ポイント: パーツ追加を繰り返せばコの字型・T字型などの複雑な形も作れます。各パーツの寸法(mm)は右パネルと描画エリアの両方で確認できます。

4. 壁・窓・ドア

壁について

  • 壁は 自動生成 されます。部屋を配置・移動・リサイズすると壁も自動で更新されます
  • 外壁（部屋の外側に面する壁）= 太い黒線
  • 内壁（部屋同士が隣接する壁）= 細い灰色線
  • 壁を手動で引く必要はありません
壁を消したい場合は、壁の元になっている部屋を移動・削除 してください。壁は部屋の境界から自動生成されるため、壁単体の削除はできません。

窓の追加

  1. 左サイドバーの 「窓/ドア」 ツールを選択
  2. 下のサブボタンで 「窓」 を選択（デフォルトで選択済み）
  3. 壁の線をクリック するとその位置に窓が配置されます
窓の記号: ═══════ （二重線）

ドアの追加

  1. 左サイドバーの 「窓/ドア」 ツールを選択
  2. 下のサブボタンで 「ドア」 を選択
  3. 壁の線をクリック するとその位置にドアが配置されます
ドアの記号: ──╮ （線 + 円弧 = 開き戸の表現） ╰─

5. 階段と1F/2F切替

階の切替

ヘッダーの 「1F」「2F」 ボタンをクリックすると表示する階が切り替わります。各階の部屋・壁・階段は独立して管理されます。

階段の追加

  1. 左サイドバーの 「階段」 ツールを選択
  2. 描画エリアで クリック＆ドラッグ して矩形を描きます
  3. 段の線と方向（UP / DN）が表示されます

階段の削除

選択ツールで階段をクリックして選択し、Delete キーで削除できます。

6. 敷地境界

  1. 左サイドバーの 「敷地」 ツールを選択
  2. 描画エリアを クリックするたびに頂点が追加 され、ポリゴンが形成されます
  3. 頂点を ドラッグ して位置を調整できます
  4. 辺の中間にある小さな丸をクリックすると 頂点を挿入 できます
スナップ解除: Alt キーを押しながら操作すると、グリッドスナップが解除されて自由な位置に配置できます。

7. 保存・読込・Undo/Redo

保存

左サイドバーの 「保存」 ボタンをクリックすると、JSON ファイルがダウンロードされます。ファイル名はプロジェクト名になります。

読込

「読込」 ボタンをクリックして、以前保存した JSON ファイルを選択すると間取りが復元されます。

Undo / Redo

元に戻す（Undo）Ctrl + Z またはヘッダーの「戻す」ボタン
やり直し（Redo）Ctrl + Y またはヘッダーの「やり直し」ボタン
